## 📖 소개

부모님이 아이와 함께 성경동화를 읽으며 신앙을 나눌 수 있도록 설계된 웹 애플리케이션입니다.

### 주요 기능
- 📚 4페이지 그림 성경동화
- 📱 반응형 디자인 (모바일/태블릿/PC)
- 📝 텍스트 크기 조절 (S/M/L)
- 📖 책장 넘기기 애니메이션
- ⌨️ 키보드/터치 제스처 지원

## 📥 설치 및 실행

### 로컬에서 실행

1. 저장소 클론
```bash
git clone https://github.com/yourusername/bible-storybook.git
cd bible-storybook
```

2. 브라우저로 열기
- `index.html` 파일을 더블클릭하거나
- Live Server 사용:
```bash
python -m http.server 8000
```

## 📝 콘텐츠 업데이트

매주 새로운 성경동화로 업데이트하는 방법:

### 이미지 교체
`assets/images/` 폴더의 파일 교체:
- `page1.jpg` - 1페이지 삽화
- `page2.jpg` - 2페이지 삽화  
- `page3.jpg` - 3페이지 삽화
- `page4.jpg` - 4페이지 삽화

### 텍스트 수정
`index.html`에서 각 `.text-page` 섹션의 내용 수정

자세한 내용은 [README.md](README.md)를 참고하세요.

## 🌐 배포

### GitHub Pages
1. Settings → Pages
2. Source를 "main branch"로 설정
3. 자동 생성된 URL 확인

### Netlify
1. [Netlify](https://www.netlify.com)에서 "New site"
2. 저장소 연결 또는 폴더 드래그앤드롭
3. 배포 완료
